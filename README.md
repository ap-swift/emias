# EMIAS Check

Консольная утилита для автоматической проверки историй болезни по требованиям
приказа Минздрава России № 530н и клинических рекомендаций.

## Возможности

- Извлечение текста из PDF-файлов историй болезни
- Разбиение документа на именованные разделы (паспортная часть, диагноз, дневники и т.д.)
- Проверка по правилам приказа 530н (обязательные разделы, реквизиты, подписи)
- Частичная проверка по клиническим рекомендациям (демо)
- Формирование наглядного HTML-отчёта

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
# Проверить один PDF
python -m emias_check /path/to/history.pdf

# Указать имя выходного файла
python -m emias_check /path/to/history.pdf --output my_report.html

# Проверить все PDF в папке
python -m emias_check /path/to/folder/
```

## Структура проекта

```
emias_check/
  cli.py          — точка входа (click CLI)
  extractor.py    — извлечение текста из PDF (PyMuPDF)
  parser.py       — разбиение текста на разделы
  rules/
    base.py       — абстрактный класс Rule + движок
    order530n.py  — правила приказа 530н
    clinical.py   — заглушки клинических рекомендаций
  report.py       — генерация HTML-отчёта (Jinja2)
  templates/
    report.html   — шаблон отчёта
```

## Ограничения MVP

- Нет автоматической загрузки из ЕМИАС
- Нет полного покрытия всех клинических рекомендаций
- Нет OCR для сканированных PDF
- Нет веб-интерфейса
